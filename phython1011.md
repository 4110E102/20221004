# Python If ... Else
## Python Conditions and If statements
```Phython
a = 33
b = 200

if b > a:
  print("b is greater than a")
```
## Elif
```Python
a = 33
b = 33
if b > a:
  print("b is greater than a")
elif a == b:
  print("a and b are equal")
```
## Else
```Python
a = 200
b = 33
if b > a:
  print("b is greater than a")
elif a == b:
  print("a and b are equal")
else:
  print("a is greater than b")
```
```Python
a = 200
b = 33
if b > a:
  print("b is greater than a")
else:
  print("b is not greater than a")
```
## Short Hand If
```Python
a = 200
b = 33

if a > b: print("a is greater than b")
```
